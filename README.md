# Luna_s2
# Features
- 🎙️ Voice Input — Continuous microphone listening via speech_recognition
- 🔊 Voice Output — Female TTS voice via pyttsx3
- 🤖 AI Responses — Local llama3.2 via Ollama for general queries
- 🚀 NASA APOD — Random astronomy picture + spoken explanation
- ☄️ Near-Earth Asteroid Tracker — Fetches and reads out asteroid proximity data from NASA NEO API
- 🌤️ Weather — Current weather conditions for Bangalore via Weatherstack
- 🧠 Flexible Keyword Matching — Triggers now activate on partial phrase matches, not just exact input
